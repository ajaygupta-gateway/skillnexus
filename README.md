# SkillNexus 🎓

AI-Powered Enterprise Learning & Development Platform — full-stack monorepo.

> **Stack:** FastAPI · React · PostgreSQL · Redis · LangChain · Docker

---

## ✨ Features

| Feature | Details |
|---|---|
| 🗺️ **Roadmaps** | Visual node-graph learning paths built with React Flow + Custom Manual Layout |
| 🤖 **AI Roadmap Generation** | Generate full roadmaps from a text prompt via LLM |
| 🔒 **Strict Mode** | All assignments require quiz pass before marking nodes done (enforced by default) |
| 🧠 **AI Tutor** | Per-node chat powered by LangChain (Groq/Gemini), with persistent history |
| 📝 **Auto Quiz** | 3-question MCQ generated by LLM; graded server-side; answers never sent to client |
| ⚡ **XP & Levels** | Event-based XP ledger: login (+5), roadmap complete (+50), quiz pass (+25), streak bonus (+30) |
| 📄 **Resume Analysis** | Upload PDF → AI extracts skills, redacts PII, suggests matching roadmaps |
| 🏆 **Leaderboard** | Global XP ranking |
| 🎛️ **Control Center** | Admin dashboard with platform stats, assignment management, analytics, and roadmap requests |
| 📊 **Analytics** | Per-roadmap completion stats, skill gap analysis, per-user progress reports |
| 🔐 **Published Lock** | Published roadmaps become read-only (no edits, node add/delete blocked) |
| 👥 **Role-Based UI** | Admins see platform management views; Learners see XP, streaks, and leaderboard |

## 🚀 Quick Start

### Prerequisites
- Docker & Docker Compose
- Node.js 18+

### 1. Backend

```bash
cd backend
cp .env.example .env
# Edit .env — add your GROQ_API_KEY (or GEMINI_API_KEY), DATABASE_URL, etc.

# Start all services (API + PostgreSQL + Redis)
docker compose up -d

# Run DB migrations
docker exec skillnexus-api sh -c "python -m alembic upgrade head"
```

API: http://localhost:8000  
Swagger docs: http://localhost:8000/docs

### 2. Frontend

```bash
cd frontend/skill-nexus
npm install
npm run dev
```

App: http://localhost:5173

## ⚙️ Environment Variables

Copy `backend/.env.example` to `backend/.env` and fill in:

| Variable | Description |
|---|---|
| `DATABASE_URL` | PostgreSQL connection string |
| `REDIS_URL` | Redis connection string |
| `SECRET_KEY` | JWT signing secret (generate with `openssl rand -hex 32`) |
| `GROQ_API_KEY` | Groq API key for LLM (quiz + AI tutor + roadmap generation) |
| `GEMINI_API_KEY` | Alternative LLM key (optional) |

> ⚠️ **Never commit `.env` to git.** It is listed in `.gitignore`.

## 🔄 Learner Progression Flow

```
Enroll in Roadmap (or get assigned by Admin)
       │
       ▼
 First Root Node Auto-Unlocked (in_progress)
       │
  [Study Node → Use AI Tutor → Read Resources]
       │
       ▼
 Click "Mark as Done" → Quiz appears (3 MCQs)
       │
    ┌──┴──┐
  Pass   Fail
   │       │
   ▼       ▼
 Node    Try Again
 Done
   │
   ▼
 All children of a parent done?
   │
   ├─► Parent has NO resources → Auto-completes ✅
   └─► Parent HAS resources → Must pass parent's quiz
   │
   ▼
 Complete all Root Sections?
   │
   ▼
 Roadmap Done (+50 XP) 🏆
```

## 🔒 Security Notes

- Progress updates require the user to be **enrolled** in the roadmap (403 otherwise)
- **All assignments enforce strict mode** — quiz required before marking any node done
- Quiz correct answers are stored **server-side only** — never returned to the client
- Resume text is **PII-sanitized** before LLM processing (phone, email, addresses, profile URLs removed)
- Passwords hashed with **bcrypt**
- Refresh tokens stored as **SHA-256 hashes** in DB (revocable)
- Access JWT lifetime: 30 min | Refresh JWT lifetime: 30 days

## 📡 Key API Endpoints

| Module | Endpoint |
|---|---|
| Auth | `POST /api/v1/auth/register` · `/login` · `/refresh` · `/logout` |
| Users | `GET /api/v1/users/me` · `PATCH /api/v1/users/me` · `GET /api/v1/users/leaderboard` |
| Roadmaps | `GET /api/v1/roadmaps` · `POST /api/v1/roadmaps` · `POST /api/v1/roadmaps/generate` |
| Roadmaps | `POST /api/v1/roadmaps/{id}/publish` · `POST /api/v1/roadmaps/request` |
| Progress | `POST /api/v1/progress/roadmaps/{id}/enroll` |
| Progress | `POST /api/v1/progress/roadmaps/{id}/nodes/{nid}` |
| Chat | `POST /api/v1/chat/sessions/{node_id}/messages` |
| Quiz | `POST /api/v1/chat/sessions/{node_id}/quiz` · `/quiz/submit` |
| Resume | `POST /api/v1/resume/upload` · `GET /api/v1/resume/me` |
| Admin | `POST /api/v1/admin/assignments` · `DELETE /api/v1/admin/assignments/{id}` |
| Admin | `GET /api/v1/admin/analytics/dashboard` · `/analytics/skill-gaps` · `/analytics/users/{id}` |
| Admin | `GET /api/v1/admin/roadmap-requests` · `PATCH /api/v1/admin/roadmap-requests/{id}` |
